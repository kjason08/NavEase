# NavEase
## 🚌 About NavEase
NavEase는 교통 이용 상황에서 발생하는 불편함을 최소화하는 경로를 제공하는 웹 서비스입니다.

NavEase가 생각하는 불편한 상황은 다음과 같습니다.
- 대중교통 이용 시 차량 내부가 혼잡한 경우
- 도로가 혼잡하여 정체된 경우
- 내가 이용할 수 없는 교통 자원으로 경로를 추천해주는 경우

NavEase는 이러한 불편한 상황들을 최소화할 수 있는 교통 경로를 사용자에게 제공해주는 것이 목표입니다.
